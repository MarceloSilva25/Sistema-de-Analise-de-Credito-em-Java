# Sistema-de-An-lise-de-Cr-dito-em-Java
Sistema de análise de crédito desenvolvido em Java com foco na aplicação de Programação Orientada a Objetos. O projeto simula o processo inicial de solicitação de empréstimo, realizando a modelagem de clientes e pedidos de crédito, aplicando conceitos como relacionamento entre objetos, encapsulamento e separação de responsabilidades.
